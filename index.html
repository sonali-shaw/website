Bovine Body Condition Scoring
Summer research done with Argonne National Labs and the UChicago Data Science Institute
A deep learning system that predicts bovine body condition scores (BCS) from aerial images to help farmers assess livestock health more consistently and objectively. Using a fine-tuned ResNet18 and multiple image filters, the classification models achieved high accuracy, outperforming regression approaches and exploring AI-assisted scoring in the absence of a ground-truth. This work explores filtering strategies, classification vs regression, and future directions for unsupervised methods to improve animal welfare and farm management.
Coloretto
Click here for code, demo, and more
This Python program brings the board game Coloretto to the terminal, letting 2–4 players draw and take colored cards to strategically build their hands and score points. It features dynamic game state management, input validation, and scoring, along with a visually appealing ASCII-style interface using rich formatting. The project highlights advanced Python skills in object-oriented design, modularity, and interactive gameplay development.
