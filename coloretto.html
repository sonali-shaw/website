Coloretto
This Python program implements a fully playable, terminal-based version of the board/card game Coloretto. Players draw and take colored cards to strategically build their hands and maximize points, with the game handling multiple players, dynamic stacks, scoring, and the final round. What makes it especially cool is its visual presentation in the terminal using ASCII-style tiles and rich formatting, creating a clear, interactive, and visually appealing interface. The code also showcases advanced programming concepts like object-oriented design, game state management, input validation, and modularity, making it both a fun game and a strong example of complex Python development.
